# Build
## In git compatible terminal:
* git clone https://github.com/SnowmanTackler/iTunesGoogleHome.git
* cd iTunesGoogleHome
* git submodule init
* git submodule update

## With Visual Studio 2015 +
* Open iTunesGoogleHome.sln
* Let VS update the websockets project
* Click run
